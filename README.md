# QRコード読み取り（スクショ・ドラッグ&ドロップ対応）

このWebページは、**スクリーンショットや画像からQRコードを簡単に読み取ることができるツール**です。  
JavaScriptで [ZXing](https://github.com/zxing-js/library) と [jsQR](https://github.com/cozmo/jsQR) の2つのライブラリを併用して、幅広いQRコードに対応しています。

## 主な機能

- **クリップボード貼り付け対応**  
  スクリーンショットを撮影後、ページで `Ctrl + V` を押すだけで自動読み取り。  

- **ドラッグ＆ドロップ対応**  
  QRコード画像をページ上のドロップエリアにドラッグするだけで読み取り可能。  

- **リアルタイム表示**  
  読み取った結果をそのままページ上に表示し、リンクがあればクリックで開くこともできます。  

- **スクリーンショット取得手順の案内**  
  Windowsでは `Windowsキー + Shift + S` を使った切り取り方法を説明、操作ミスを防ぎます。  

## 使用方法

1. ページを開く  
2. QRコード画像を用意する  
   - スクリーンショットを撮影する  
   - 画像ファイルを用意する  
3. 以下の方法でQRコードを読み取る  
   - **貼り付け**: `Ctrl + V` を押す  
   - **クリップボード読み取りボタン**をクリック  
   - **ドラッグ＆ドロップ**で画像をドロップ  
4. 結果がページに表示されます  

## ボタンの説明

- 📋 **クリップボードから読み取る**: 画像がクリップボードにある場合に読み取り  
- 📌 **貼付けを待機して開始**: `Ctrl + V` を待機して貼り付け可能  
- 🔄 **リセット**: ページを初期状態に戻す  

## 技術スタック

- **HTML / CSS / JavaScript**  
- **ZXing / jsQR**: QRコード解析ライブラリ  
- **ブラウザのみで動作（サーバ不要）**  

## 注意事項

- 画像サイズが大きい場合、最大800pxにリサイズして解析されます。  
- まず `jsQR` で解析し、失敗した場合に `ZXing` で再解析します。  
- Windows・Mac 両方のスクリーンショットに対応しています。  

---
